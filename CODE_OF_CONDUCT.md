# Code of Conduct

This project expects respectful, constructive participation.

## Expected Behavior

- Be kind and professional.
- Give clear technical feedback.
- Respect different experience levels.
- Keep hardware safety discussions careful and specific.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Publishing private information.
- Encouraging unsafe machine operation.
- Spamming issues or pull requests.

## Enforcement

Maintainers may edit, hide, or remove comments and may restrict participation when needed to keep the project safe and productive.
